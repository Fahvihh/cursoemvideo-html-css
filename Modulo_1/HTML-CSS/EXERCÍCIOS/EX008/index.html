Principais Formatações

Negrito / Destaque

Tag B -> Em HTML podemos utilizar a tag b para deixar em negrito.
Exemplo: Olá, Mundo!

Strong -> Também podemos usar a tag Strong para deixar em negrito porém ela é semântica, ou seja, tem o siginificado por trás que seria texto em destaque.
Exemplo: Você sabia que 43% da população espirra quando olha para o sol? (exemplo fictício)

Itálico / Ênfase

Tag I -> podemos utilizar a tag i para deixar em itálico um texto. (Não semântico)
Exemplo: Bom dia a todos!!

EM -> Também podemos usar a tag EM para deixar em itálico porém ela é semântica, ou seja, tem o siginificado por trás.
Exemplo: Atualmente moro em Belo Horizoente. É a capital de MG.

Texto Marcado

Podemos criar um texto marcado usando a tag mark

Texto Grande x Texto Pequeno

Parar gerar um texto grande usamos a tag big e para deixar um texto pequeno usamos a tag small

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Existe também o sublinhado com a tag U (não semântica)

Texto sobrescrito

Para inserir coisas do tipo x20+3

Texto subscrito

Para inserir coisas do tipo H2O
